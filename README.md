# garden-notes

The markdown notes (Obsidian vault) behind **[leahtu.github.io/garden](https://leahtu.github.io/garden)**.

- Open this folder as a vault in **Obsidian** and write notes here.
- On push, the site repo ([leahtu/leahtu.github.io](https://github.com/leahtu/leahtu.github.io)) rebuilds the garden and publishes it.

Write in markdown, link notes with `[[wikilinks]]`, and organize with `#tags`.
